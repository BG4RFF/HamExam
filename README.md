# Ham考试

这是一个帮助业余无线电技术能力认证考试的软件，用来背题。

软件含有题库：

* 国内考试的A、B、C三级题库（为170418版本，由170418版本的题和140331版本的附图构成）
* FCC Technician: 2014-2018
* FCC General: 2015-2019
* FCC Extra: 2016-2020

提供学习、练习错题和模拟考试三个主要功能。
